# sd-plugins

Monorepo for Smart Data Claude Code plugins.

## Plugins

| Plugin                        | Description                                                                                                                    |
| ----------------------------- | ------------------------------------------------------------------------------------------------------------------------------ |
| [sd-harvest](sd-harvest/)     | Harvest time tracking — log daily work, assemble weekly timesheets, submit to Harvest                                          |
| [sd-team](sd-team/)           | Team availability and suggestion — view utilization from Harvest, manage roster via Google Sheets, suggest team configurations |
| [sd-estimates](sd-estimates/) | Project estimate generator — produce tiered cost proposals using SD bill rates and team configurations                         |

## Getting Started

Each plugin lives in its own directory with its own README, setup instructions, and dependencies. See the individual plugin folder for details.
